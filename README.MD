# 🦅 UNUL - The Omnidisciplinary Language
**Fase 2 (Stable) - Python Prototyping Ecosystem**

Selamat datang di repositori resmi **UNUL (Unifed Native Universal Languange)**, bahasa pemrograman tingkat tinggi beraksen Nusantara yang dirancang khusus sebagai bahasa fundamental (Native Language) untuk sistem operasi **LinuxDNC** dan pembuatan filesystem **DUFS**.

Bahasa ini diciptakan oleh **Arsitek LinuxDNC** di bawah payung "Wisata Masa Lalu", dengan mengusung filosofi *Omnidisciplinary*—satu bahasa untuk menjembatani sistem kernel, web, IoT, hingga API tingkat tinggi.

Saat ini UNUL berada di **Fase 2**, di mana ekosistem utamanya (Lexer, Parser, dan Interpreter) dibangun dan distabilkan di atas mesin Python sebelum nantinya diterjemahkan menjadi C murni di Fase 3.

## 🌟 Fitur Utama (Fase 2)
- **Sintaks Human-Readable:** Menggunakan bahasa Indonesia yang mengalir (Takdir, Karma, Siklus).
- **Omni-Data:** Mendukung Teks, Angka, Array, hingga Objek/JSON murni secara bawaan.
- **Memory Protection:** Menjerit *Kernel Panic* saat mendeteksi Array Statis yang melampaui batas kavling memori.
- **Time-Travel (Mesin Waktu):** Memiliki fitur bawaan `balikan` untuk membatalkan takdir (variabel) ke kondisi masa lalunya.
- **Protokol Utusan:** Bisa melakukan HTTP GET (`ambil`) dan POST (`setor`) secara bawaan tanpa pustaka eksternal!

## 📂 Struktur Ekosistem
Ekosistem UNUL dibangun dengan tata letak kelas dunia:
- `/bin`: Berisi eksekutor CLI (`unul`) untuk menjembatani sistem operasi dengan mesin.
- `/src/core`: Jantung dari UNUL (Trinitas Enki: Lexer, Parser, Eksekutor).
- `/tests`: Ruang isolasi pengujian per-fitur untuk memastikan mesin bebas *bug*.
- `/example`: Kumpulan *showcase* aplikasi utuh yang dibangun dengan UNUL & UNLL.

## 🚀 Instalasi CLI

### Pengguna Linux / MacOS
Buka terminalmu dan jalankan mantra sakti ini:
```bash`
`chmod +x install.sh`
`./install.sh`
`source ~/.bashrc`

### Pengguna Windows
Pengguna Windows Cukup klik ganda (double-click) pada file install.bat, lalu restart terminal/CMD kamu.

### Cara Menjalankan Program
Setelah instalasi sukses, kamu bisa menjalankan kitab takdir (.unul) di mana saja:

`unul ayo nama_file.unul`
